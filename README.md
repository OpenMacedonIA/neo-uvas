**The Skills Layer of WatermelonD.**

BlueberrySkills is a collection of atomic, powerful functions scattered throughout the system. Just like blueberries, they are small but packed with utility.

## Included Skills
- **System**: Control system services and hardware.
- **Network**: Manage WiFi and connectivity.
- **Media**: Control playback and volume.
- **Organizer**: Calendar, alarms, and reminders.
- **SSH**: Remote connection management.
- **Finder**: Locate files and resources.

## Developing Skills
To create a new skill, add a new python module in this directory and register it in `WatermelonD/NeoCore.py`.
